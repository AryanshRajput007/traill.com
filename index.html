Basic Terminologies of DBMS

  • Data: Data refers to raw facts, information, or statistics that are collected, stored, and processed by computers and other information systems.
    Data: 35, 40, 50, 60
  • Information: Information is the processed and meaningful representation of data.
    Information: Average age of a group of people is 46.25
  • Metadata: Metadata is data that provides information about other data. It makes it easier to understand, manage, and use data effectively.
  • Data Dictionary: A data dictionary is an information repository that contains metadata.
  • Data Warehouse: A data warehouse is an information repository that stores data.
  • Field: A field is a character or group of characters that have a specific meaning. It can be thought of as a column in a spreadsheet or a cell in a table.
  • Record/Tuple: A record, also known as a row, is a complete set of related data within a database table.
  • Data Abstraction: Data abstraction simplifies interactions with complex things (like databases or computer programs) by using easy-to-understand commands or interfaces, without worrying about the intricate details of how they work behind the scenes.
